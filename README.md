Автономное интерактивное веб-приложение для проверки знаний и комплексной подготовки к экзаменам по русскому языку. Включает 5 полноценных вариантов по 20 тестовых заданий в каждом.

Проект разработан без внешних зависимостей и фреймворков. Работает «из коробки» прямо в браузере.

---

## ✨ Особенности проекта

* **🗂️ 5 тематических вариантов:** суммарно 100 уникальных тестовых заданий.
* **🎯 Профильные разделы:** автоматическая группировка вопросов по 5 ключевым направлениям лингвистики с цветовой индикацией:
  * **Орфография** — правила написания слов, чередование, суффиксы.
  * **Пунктуация** — обособленные члены, прямая речь, сложные предложения.
  * **Лексикология** — синонимы, антонимы, паронимы, устойчивые выражения.
  * **Стилистика** — тропы, стили речи, эмоциональная окраска.
  * **Морфология** — грамматические категории, падежи, разряды частей речи.
* **⚡ Мгновенная проверка:** подсчет результатов по 100-балльной системе (по 5 баллов за верный ответ).
* **📊 Наглядный фидбек:** подсветка правильных и ошибочных ответов зеленым и красным цветами, учет пропущенных вопросов и вывод финального вердикта.
* **🎨 Адаптивный дизайн:** интерфейс корректно отображается как на ПК, так и на мобильных устройствах, используя системные шрифты и CSS-переменные.
